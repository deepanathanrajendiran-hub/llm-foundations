LLM Foundations

A local, static teaching series for LLM / GenAI interview prep. Each page is a deep, self-contained lecture — built from the question bank in ../interview-prep/ — that walks a topic from first principles to current SOTA, with diagrams, math, code, and a comparison cheat-sheet of every approach you should know.

Phase 1 · Foundations — built & ready to teach

One module, split into an overview + four focused pages (~1 hr total). Start with the overview.

START HERE · OVERVIEW

Transformer Foundations — Overview

The whole pipeline in ~10 min: text → tokens → vectors → attention → block → position. Links to the four deep dives.

~10 min MODULE 02

Tokenization & Embeddings

Word → char → BPE → byte-level → WordPiece → Unigram → tokenizer-free; static vs contextual embeddings. Live tokenizer demo.

~12 min MODULE 04

Attention Mechanisms

Scaled dot-product, multi-head, MQA/GQA/MLA, FlashAttention, sliding-window/sparse/linear, attention sinks. Heatmap demo.

~15 min MODULE 03

Transformer Architecture

The block (attention + FFN + residual + norm), pre-norm/RMSNorm/SwiGLU, BERT vs GPT vs T5, parameter counting & scaling.

~13 min MODULE 05

Positional Encoding & Long Context

Sinusoidal, learned, relative/T5, RoPE, ALiBi, NoPE; context extension (PI/NTK/YaRN/LongRoPE); lost-in-the-middle & sinks.

~14 min

Phase 2 · Training & Alignment — built & ready to teach

From a raw pretrained base to an aligned assistant. These four are reveal.js slide decks — click to present (←/→ nav · S speaker notes · O overview · F fullscreen); append ?print-pdf to the URL to export a PDF.

MODULE 06 · ▶ DECK

Pretraining & Scaling Laws

Self-supervised objectives (causal LM, MLM, span corruption, UL2, FIM) and the scaling laws — Kaplan, Chinchilla, compute-optimal — that size a run.

~16 min MODULE 07 · ▶ DECK

Fine-tuning & PEFT

SFT and instruction tuning, then the parameter-efficient toolkit: adapters, prefix/prompt tuning, LoRA, QLoRA, DoRA, IA³. With a LoRA demo.

~17 min MODULE 08 · ▶ DECK

RLHF & Alignment

Reward modeling + PPO, the DPO shortcut, critic-free GRPO, and AI / verifiable feedback (RLAIF, RLVR). With a PPO-loop demo.

~18 min MODULE 31 · ▶ DECK

Synthetic Data & Data-Centric AI

The data wall, generation (Self-Instruct, Evol-Instruct, distillation), self-improvement (STaR), verification, and model collapse. With a flywheel demo.

~17 min
